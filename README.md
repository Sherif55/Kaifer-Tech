# Kaifer Tech

A static portfolio website for Kaifer Tech built with modern HTML, CSS, and JavaScript.

## Overview

This project is a one-page landing site showcasing services, portfolio items, and a contact form. The site includes:

- responsive hero, about, services, work, and contact sections
- animated project cards and content reveal effects
- custom cursor interaction
- contact form that opens the user's email client with a pre-filled message
- simple project description containers for easy updates

## Files

- `index.html` — main site markup
- `styles.css` — all page styling and responsive layout rules
- `script.js` — interactive site behavior and contact form handling

## Updating the site

- Change project details in `index.html` inside the `.project-content` blocks.
- Update the contact section copy and form as needed in `index.html`.
- Edit the theme and layout in `styles.css`.
- Add or change interactivity in `script.js`.

## Local preview

Open `index.html` in your browser to preview the site locally.
